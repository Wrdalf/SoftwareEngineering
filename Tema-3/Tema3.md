## Лабораторная работа №7

### Напишите программу, в которой вы наглядно посмотрите, как работает цикл for проходя в обратном порядке, то есть, к примеру не от 0 до 10, а от 10 к 0. В уже готовой программе показано вычитание из 100, а вам во время реализации программы будет необходимо придумать свой вариант применения обратного цикла

### Код программы

```python
def main(amount, number, step):
    if step == False:
        for i in range (amount , 0, -1):
            print(f"{number} * {i} = {number * i}")
    else:
        for i in range (amount):
            print(f"{number} * {i} = {number * i}")

number = int(input('таблицу умножения какого числа вы хотите увидеть? \n'))
amount = int(input('До какого числа нужно умножать? \n'))

step = input('Выберите в какой последовательности вывести таблицу умножения\n'
             'Если хотите чтобы таблица выводилась в порядке возрастания напишите "+"\n'
             'Если хотите чтобы таблица выводилась в порядке убывания напишите "-"\n')
if step == '+':
    main(amount, number, True)
elif step == '-':
    main(amount, number, False)
else:
    main(amount, number, True)

```
