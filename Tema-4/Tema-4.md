# Самостоятельная работа №1

#### дайте подробный комментарий для кода, написанного ниже, комментарий нужен для каждой строчки кода, нужно описать что она делает. Не забудьте, что функции комментируются по-особенному

```python
from datetime import datetime # Подключаем модуль datetime из библиотеки
from math import sqrt #  импорт из пакета math функции sqrt


def main(**kwargs):
    """
    Поиск корня произведения первого элемента массива со вторым элементом массива

    Args:
        **kwargs (key [int,int])

    returns:
        result (float): корень произведения

    """
    for key in kwargs.items(): # проходим циклом по кортежу
        result = sqrt(key[1][0] ** 2 + key[1][1] ** 2) # возводим первый и второй элементы массива и суммируем их
        print(result) # выводим результат

if __name__ == '__main__': # точка входа
    start_time = datetime.now() # переменной start_time присваивается функция datetime.now()
    main(  # вызов функции
        one = [10, 3],  # аргументы передаваемые в функцию
        two = [5, 4],
        three = [15,13],
        four = [93, 53],
        five = [133, 15]
    )

    time_costs = datetime.now() - start_time # рассчет затраченого времени на работу программы
    print(f"время выполнения программы - {time_costs}") # вывод времени выполнения
    
```

#### в программе импортируется два модуля, datetime и sqrt, из библиотеки Python. функция main, которая принимает аргумент **kwargs. Этот аргумент ожидается быть словарем, где ключи - это строки, а значения - это списки из двух целых чисел. Внутри функции main есть цикл for, который перебирает элементы словаря kwargs. Каждый элемент имеет вид (ключ, значение). Для каждого элемента программа выполняет следующие действия:Возводит первый элемент списка (первое число) в квадрат.Возводит второй элемент списка (второе число) в квадрат. Затем программа вызывает функцию main, передавая аргументы в виде именованных аргументов, где каждое имя - это ключ, а значение - это список из двух чисел. Замеряет время выполнения программы с помощью модуля datetime. Выводит результаты вычислений и время выполнения программы.

#### также **kwargs (как и *args) - это мощный механизм в Python для работы с аргументами функции. kwargs является сокращением от "keyword arguments" и представляет собой словарь, который позволяет передавать именованные аргументы в функцию. Каждый элемент словаря имеет имя (ключ) и значение.

```python
def calculate_average(numbers):
    if not numbers: # проверка на то, что массив не пустой, чтобы не было деления на 0
        return None

    total = sum(numbers)
    average = total / len(numbers)
    return average


if __name__ == "__main__":
    print('* Введите хотя бы одно число * ')
    input_numbers = []

    while True:
        user_input = input("Введите числовый аргумент (или 'end' для завершения ввода): ")

        if user_input.lower() == 'end':
            break

        try:
            number = float(user_input)
            input_numbers.append(number) # ловим числа из пользовательского ввода

        except ValueError:
            print("Неверный формат числа. Попробуйте ещё раз.") # ловим исключения при вводе неверного аргумента

    average = calculate_average(input_numbers)  # Вычисляем среднее арифметическое

    if average is not None:
        print(f"Среднее арифметическое: {average}")
    else:
        print("Вы не ввели ни одного числа")
```

2. **Возвращение значений**: Функции могут возвращать значения с помощью ключевого слова `return`. Если `return` не указан, функция возвращает `None`.

3. **Аргументы функций**:
   - **Обязательные аргументы**: Функции могут иметь обязательные аргументы, которые должны быть переданы при вызове.
   - **Аргументы по умолчанию**: Функции могут иметь аргументы, которым назначаются значения по умолчанию.
   - **Аргументы переменной длины**: Функции могут принимать переменное количество аргументов с помощью `*args` (позиционные аргументы) и `**kwargs` (именованные аргументы).

4. **Область видимости (scope)**: Переменные, определенные внутри функции, имеют локальную область видимости и не видны вне функции. Глобальные переменные могут быть доступны внутри функции, но если требуется изменить глобальную переменную, необходимо использовать ключевое слово `global`.

5. **Рекурсия**: Функции могут вызывать сами себя, что называется рекурсией.

6. **Импорт функций**: Функции могут быть определены в других модулях и импортированы для использования в других программах.

**Стандартные модули/библиотеки в Python**:

1. **math**: Модуль `math` предоставляет функции для математических операций, такие как вычисление корня, тригонометрические функции и др.

2. **datetime**: Модуль `datetime` используется для работы с датой и временем. Он позволяет создавать, форматировать и выполнять операции над датами и временем.

3. **random**: Модуль `random` используется для генерации случайных чисел и выполнения различных операций, связанных со случайными событиями.

4. **sys**: Модуль `sys` предоставляет доступ к системным переменным и функциям, таким как `sys.argv` для работы с аргументами командной строки.

5. **os**: Модуль `os` используется для выполнения операций, связанных с операционной системой, такие как работа с файлами, директориями и окружением.

6. **json**: Модуль `json` позволяет работать с данными в формате JSON, что полезно для обмена данными между программами.

7. **requests**: Модуль `requests` используется для отправки HTTP-запросов и работы с веб-ресурсами.

8. **re**: Модуль `re` предоставляет возможность работы с регулярными выражениями для поиска и обработки текстовой информации.

9. **csv**: Модуль `csv` предоставляет функциональность для работы с файлами в формате CSV (Comma-Separated Values), что полезно при обработке таблиц данных.

Стандартные модули и библиотеки в Python обеспечивают множество возможностей для решения различных задач, и их использование позволяет значительно упростить разработку приложений и снизить затраты времени на написание кода.
